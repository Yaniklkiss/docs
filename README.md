# Тестовая документация
Для веб-приложения интернет-магазина электроники ***** необходимо оформить тестовую документацию. На сайте представлен каталог товаров, информация о фирме, контакты. Реализованы функции регистрации и авторизации пользователя, фильтрации и сортировки товаров в каталоге. Неавторизованный пользователь не может перейти на вкладки "Корзина", "Оплата" и "История заказов". Авторизованный пользователь может добавить выбранные товары в корзину и оформить заказ. Оплата заказа возможна картами MasterCard и VISA, а также через PayPal. После оплаты заказа он появляется на странице "История заказов". Также есть требования по каждому функциональному модулю.
